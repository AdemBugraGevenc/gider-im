# HesApp.im - Kişisel Finans Takip Uygulaması

## Proje Hakkında
HesApp.im, kullanıcıların gelir ve giderlerini takip edebileceği, tasarruf hedefleri oluşturabileceği ve finansal durumlarını grafiklerle analiz edebileceği modern bir web uygulamasıdır. Mobile-first (mobil öncelikli) yaklaşımıyla tasarlanan arayüz, hem mobil cihazlarda hem de masaüstü tarayıcılarda kusursuz bir deneyim sunar.

## Özellikler
- **Cüzdan Yönetimi:** Gelir ve gider kayıtlarını hızlıca ekleyin, kategorilere ayırın.
- **Zaman Çizelgesi:** Harcamalarınızı gün bazında gruplandırılmış bir akışta görüntüleyin.
- **Tasarruf Hedefleri:** Hayalinizdeki tatil veya yeni bir bilgisayar için hedef oluşturun, birikim yapın.
- **Detaylı Analiz:** Aylık gelir/gider dağılımını ve kategori bazlı harcamalarınızı grafiklerle inceleyin.
- **Karanlık Mod:** Göz yormayan modern karanlık tema desteği.
- **Veri Gizliliği:** Tüm verileriniz tarayıcınızda (LocalStorage) saklanır, dış sunuculara gönderilmez.

## Teknolojiler
Bu proje, modern web teknolojileri kullanılarak geliştirilmiştir:
- **Core:** React 18, TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Package Manager:** pnpm
